Serophin is a meditation application that is built on BroccolipApps framework.

It is a mobile only application for now. It will be available on iOS and Android. Using bare react native.

## Features

### Meditation

User can choose

- Guided vs unguided meditation
- Haptic feedbacks
- Meditation duration
- Background sounds (nature, rain, waves etc)
- Background music or noise

### Breathing

User can choose

- Breathing pattern
- Breathing duration
- Background sounds
- Background music or noise
- Haptic feedbacks

### Sleep

User can choose

- Sleep sounds
- Sleep duration
- Background sounds
- Background music or noise

### Meditation course

This is an in-app course that consists of 10x3 courses. Level 1, 2, 3. Each course has 10 lessons. Each lesson is 10 minutes long.

It is a one time in-app purchase for $10 USD for all 3 levels. Course levels are not sold separately.

### Settings

- System/Dark/Light theme
- Signout

### Sign in/up

- Email (magic link)
- Sign in with Google
- Sign in with Apple

### What it is not

- No gamification, streaks etc
- No notifications
- No sleep tracking
- No meditation tracking
- No breathing tracking
